The hypothesis asserts that ρ21 = ρ31. Since ρ32 is not equated to anything, it will get a null-consistent estimate but it will not affect the p value for the test.

In this case, since ρ21 = 0.109 and ρ31 = 0.021 are equal under the null, the OLS estimate of ρ21 = ρ31 is (0.109 + 0.021)/2 = 0.065. The OLS estimate of ρ32 is the sample value, since it wasn't equated to anything under the hypothesis.

Next we have either the GLS, TSGLS, ADF, or TSADF estimates, depending on which method was selected. Here are the TSGLS estimates:

1   0.077   0.203   [-0.381, 0.504]  
2   0.065   0.149   [-0.391, 0.495]  

If ρ21 = ρ31 then the TSGLS estimate of ρ21 = ρ31 is 0.065 and that for ρ32 is 0.077. This table also gives confidence intervals on those point estimates, using a strict Bonferroni on an alpha of .05 via the Fisher transform method.

Next we have the "Significance Test Results":


Chi Square  df  plevel
0.101   1   0.75  

This indicates that the probability of drawing this sample if the null is true is 75%.

Next, since we used raw data, we have the "Assessment of Multivariate Normality". The specific test will depend on 1) whether there is missing data and 2) which deletion method was chosen. Yuan, Lambert & Fouladi's (2004) is given if pairwise deletion is employed, and Mardia's (1970) is provided otherwise. Yuan, Lambert & Fouladi's (2004) test is only useable if the observed marginals do not sit in a restricted range, so a test of that assumption is provided for each variable with missing data. This is only possible if there is at least one variable without missing data.

Since the example data is complete, Mardia's (1970) test is used. Here is the test of multivariate skewness:


Group Multivariate Skewness Chi Square df     plevel
1   27.801   115.838   56   < 0.001  

And here is the test of multivariate kurtosis:


Group Multivariate Kurtosis Z statistic plevel (2-tailed)
1   55.493   2.854   0.004  

Since the p value is low for both, multivariate normality is not supported and we should rerun the test using the TSADF method.

Suppose we remove one value from the data set and choose pairwise deletion. Now Yuan, Lambert, and Fouladi's (2004) test is displayed. This begins with an assessment of the distribution of the observed marginals:

1   1   1   0.416   0.677  

The p value is high, so the observed marginals do not appear to be restricted and we can move on to the assessment of multivariate normality:


Group       M2 plevel (two-tail)
1   2.475   0.013  

Again, the p value is low, so multivariate normality is not supported and we should re-run the test using TSADF.
