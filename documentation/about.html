MML-WBCORR is an extension and web application implementation of WBCORR, a correlation pattern hypothesis test program developed for Mathematica by Steiger (2004). It can handle raw or correlation data, in one or more samples, with or without equal sample sizes, and with or without the assumption of multivariate normality.

This About page is in development as MML-WBMCORR has several additional functionalities from the program developed by Steiger (WBCORR, 2004). These additional functionality and should be described here.

Please also refer to http://www.sfu.ca/psychology/research/mml/resources.html

MML-WBCORR may be cited in APA style as follows:

Fouladi, R. T., & Serafini, P. E. (2018). MML-WBCORR. Retrieved from http://www.sfu.ca/psychology/research/mml/resources.html

Offline and bootstrap versions of MML-WBCORR are available at its github page.
